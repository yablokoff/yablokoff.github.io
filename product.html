Top features

Flexible Material Setup and Pricing Parameters
The user-friendly and intuitive interface provides a clear layout of changeable variables related to setting up materials for the end user to choose from inside the widget and the pricing parameters associated with them, including also such broadly applicable common denominators as startup costs and minimal order size.
End-User Authentication
The widget makes it possible to track visitors to the site of 3D printing professionals and effectively following up on end user leads in a very targeted manner, thus increasing the lead-to-order conversion rate.
100% Secure AWS Cloud
The DigiFabster solution is cloud-based, utilizing the most reputable servers at Amazon Web Services (AWS) to provide peace of mind and reliable assurances of security and integrity of all data.
Widget customization
The customer can request, and promptly receive, a wide variety of customizations of the widget itself, as well as ways the end user receives and sees the written communication, with the ongoing support of most widely used languages of the world.
